## ðŸš€ **Available Endpoints**

### 1. **Generate Single Text Audio**
```bash
POST /api/v1/tts/generate-text
```
**Request Body:**
```json
{
  "text": "Hallo! Willkommen zu KlarText.",
  "voiceName": "Sulafat",
  "language": "de",
  "temperature": 1
}
```

### 2. **Generate Lesson Audio**
```bash
POST /api/v1/tts/generate-lesson/{lessonId}
```
**Request Body:**
```json
{
  "voiceName": "Sulafat",
  "language": "de", 
  "chunkSize": 300
}
```

### 3. **Generate Batch Audio**
```bash
POST /api/v1/tts/generate-batch
```
**Request Body:**
```json
{
  "texts": ["der Hund", "die Katze", "das Haus"],
  "voiceName": "Aura",
  "language": "de"
}
```

### 4. **Get Supported Voices**
```bash
GET /api/v1/tts/voices
```

### 5. **Get Supported Languages**
```bash
GET /api/v1/tts/languages
```

### 6. **Play Audio Files**
```bash
GET /api/v1/tts/audio/{fileName}
GET /audio/gemini/{fileName}  # Direct static access
```
